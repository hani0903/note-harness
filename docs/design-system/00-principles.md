# 원칙 — Creative North Star

## "The Curated Archive"

개인 박물관의 소장품처럼 모든 TIL 항목을 다룬다.

- **Soft Minimalism** — 쉬운 SaaS 템플릿 톤을 거부한다.
- **Intentional Asymmetry + Tonal Depth** — 딱딱한 보더 박스 대신, 중립 톤의 레이어 위에 콘텐츠가 떠 있는 구조를 만든다.
- **편집자의 침묵** — UI가 인사이트와 경쟁하지 않는다. 조용한 학습의 순간을 우선한다.

---

## The "No-Line" Rule

섹션 구분에 `1px solid border`를 사용하지 않는다.

경계는 오직 **배경 색상 시프트**로만 표현한다.

| 영역     | 토큰                    |
| -------- | ----------------------- |
| 사이드바 | `surface_container_low` |
| 본문     | `surface`               |

경계는 느껴지지만, 드러나지 않아야 한다.
