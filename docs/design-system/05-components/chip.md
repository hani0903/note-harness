# Knowledge Token (Chip / 태그)

토픽 태그, 분류 칩 등에 사용한다.

```
#javascript   #design
```

| 속성   | 값                                      |
| ------ | --------------------------------------- |
| 배경   | `surface_container_highest` (`#dbe4e7`) |
| 텍스트 | `on_surface_variant` (`#586064`)        |
| 모서리 | `full` (pill)                           |

## 원칙

- border 추가 금지.
- 선택된 칩과 일반 칩의 구분은 배경 토큰 시프트로만 표현한다.
- 텍스트 색상을 강조색(`tertiary`)으로 바꾸지 않는다.
