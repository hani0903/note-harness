# 스페이싱 토큰

Tailwind CSS v4의 기본 spacing 스케일을 사용한다 (1 unit = 0.25rem).

## 사용 예시

| 토큰        | 값        | 용도                            |
| ----------- | --------- | ------------------------------- |
| `spacing.1` | `0.25rem` | Input label과 input 사이 gap    |
| `spacing.4` | `1rem`    | 리스트 항목 간격 (Divider 대체) |

## 원칙

- 리스트 항목 사이 선 대신 `spacing.4` 간격으로 구분한다.
- 컴포넌트 내부 요소 간격은 `spacing.1` ~ `spacing.2` 범위를 기본으로 한다.
