# DON'T — 하지 말아야 하는 것

- `border: 1px solid` 로 섹션 구분 (No-Line 규칙 위반)
- `box-shadow` 남용 (카드, 리스트 항목에 적용 금지)
- 지나친 컬러 대비
- 카드마다 `box-shadow` 적용
- 정보보다 UI가 더 눈에 띄는 구조
- "템플릿 SaaS 느낌"의 평면적인 단색 버튼
- 리스트 항목 사이에 `<hr>` 또는 border-bottom 사용
- 하드코딩 색상 사용 (`#000`, `#000000`, `gray` 등 — 반드시 디자인 토큰 사용)
- Secondary / Ghost 버튼에 border 추가
- Chip에 border 또는 강조색 텍스트 사용
