# Note App Design System — The Digital Atelier

> UI·CSS·컴포넌트 작업 시 반드시 이 문서를 진입점으로 삼는다.
> 세부 규칙은 아래 파일을 참조한다.

---

## 핵심 원칙 한 줄 요약

- **No-Line**: 선(border) 대신 배경 색 시프트로 경계를 표현한다.
- **Tonal Layering**: 그림자 대신 배경 레이어 차이로 깊이를 만든다.
- **Quiet Interface**: UI가 콘텐츠보다 눈에 띄어선 안 된다.
